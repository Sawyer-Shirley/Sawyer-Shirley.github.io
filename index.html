Timeline

How I got started.

  • Summer 2018

    My Decision

    During the summer of 2018, I was in New York City visiting the college that I originally planned on going to. It was a culinary-focused school. However, a teacher from my school, Justin Browning, invited me to try this brand-new program he was beginning with the help of a tech company it was called Interapt Skills.

  • August 8, 2018

    Interapt Begins

    Interapt began the first half of my senior year in high school. It is the program that introduced me to the world of coding and also taught me how to progress on my own.

  • October 9, 2018

Pixel

A video game library

This project allowed users to enter a game and its description into a cell and gives it a due-by date. The user then can change the game status from "checked out" to "checked in" when they return the game which then removes the due-by date.

  • Date: October 2018
  • Category: Convenience

Pop!

A task manager

Pop! is a task manager that allows users to enter a task they should complete with some details about said task and mark it as a priority. The task is then stored in a list where the user can delete the task or change its completion status.

  • Date: September-December 2018
  • Category: Convenience
